This project analyzes Spotify playlists by treating music as data.

and recommends similar tracks using machine learning.

## Tech Stack
- Python
- Spotify Web API
- Pandas / NumPy
- Scikit-learn
- Matplotlib

## Features
- Playlist audio feature analysis
- Data visualization
- Cosine similarity-based recommendations

## Why This Project?
This project explores the intersection of music and technology by applying
data analysis and machine learning to creative content.
